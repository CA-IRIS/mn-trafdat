This site is for the distribution of traffic data collected by the IRIS traffic management system.

There are several types of data requests which can be made. Some parameters for the requests are did, year, date, sid and ext.

param Description
did District ID of detection system
year 4-digit year
date 8-digit date (yyyyMMdd)
sid Sensor ID
ext Extension (v30, c30, s30, vmc30, vs30, vm30, vl30, pr60, pt60)
Request form Description Content type
/ Documentation (this page) text/html
/did/year.json Get dates with sample data application/json
/did/year/date Get sensors sampled on a date application/json
/did/year/date/sid.ext Get raw sample data for a sensor application/octet-stream
/did/year/date/sid.ext.json Get sample data for a sensor application/json
/did/year (deprecated) Get dates with sample data text/plain
/year (deprecated) Get sampled dates; "tms" did text/plain
/year/date (deprecated) Get sensors sampled; "tms" did application/json
/year/date/sid.ext (deprecated) Get raw sample data; "tms" did application/octet-stream
